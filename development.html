Software Development Projects

Here are some projects that I have done and others I am working on to showcase my abilities as a software engineer.

As mentioned in my about page, I work in Javascript utilizing NodeJS, ReactJS, EJS and mongo primarily for building apps and sites. I will be adding more apps/projects as I continue honing my development skills. But these will serve as at least a baseline of some proficiency.

React Youtube APP

This app is built using React and uitilizes Axios to make calls to the Youtube API. This app was also a great way to learn to deal with state and how to handle it.

React Pictures Search App

This app will search through Unsplash.com and returns up to 10 results of whatever you search. It utilizes state, async await, params and Unsplash's API.

React Todo App

This is a simple react app for a todo list for creating new todos and allowing you to delete those that are completed.

Heroku Deployed EJS Simple Webpage

This is a very basic webpage built using Node, express and EJS. Click the picture to use the app live in heroku. (And you can disregard the ridiculous wording. I was being a goof.)

I am currently working on learning deployment with Heroku (it's a real pain!!). So far I am troubleshooting issues related to deploying React applications, but I did manage to launch this Node/EJS page

Javascript Drum Kit

This app uses only Javascript, html and css to play sounds and effects mapped to keys on the keyboard allowing the user to make some music with a "drum kit". This app is part of the Javascript 30 Course from Wes Bos as a coding challenge to build 30 javascript projects in 30 days.
