Projects displayed here all utilize Javascript, HTML and CSS. Many of them also utilize Bootstrap (or Semantic UI), NodeJS, ReactJS, EJS and MongoDB for building apps and sites. I will be adding more apps/projects as I continue honing my development skills.

JS Video Speed Changer

Added 12/21/19

This is a simple page with a video player where the user can adjust the playback speed of the video using the bar to the right of the player. It uses numerous querySelectors and a more complicated addEventListener with numerous pieces to make the video speed adjust and display the current playback speed. It can go faster or slower than base (1.0x) speed. This is a project from the Javascript 30 Course for 30 javascript projects to build in 30 days.
